About

Simple douban oauth2是一个非常简洁的豆瓣PHP Oauth2 SDK，为小型PHP网站提供了简单的豆瓣Oauth2认证及豆瓣各类Api调用接口。目前还有很多api接口没有经过测试，我会尽快完成测试，并逐步完善PHP客户端文档。

Usage

这里提供了一个使用Simple douban oauth2来发送图片广播的例子。

Simple douban oauth2的所有API类都保存在api文件夹中，你可以非常方便的按照需求修改各个接口。

一个简单的API类示例：

class Book extends Base {

    // 必须包含构造函数
    public function __construct($clientId, $accessToken = null)
    {
        parent::__construct($clientId, $accessToken);
    }

    public function getBook($id)
    {
        $this->uri = '/v2/book/'.$id;
        return $this;
    }
}

添加无需授权的GET请求：

public function get($id)
{
    $this->uri = '/v2/user/'.$id;
    return $this;
}

添加需授权的GET请求：

public function me()
{
    $this->uri = '/v2/user/~me';
    // 设置Authorization header
    $this->header = $this->authorizeHeader;
    return $this;
}

添加需授权的POST请求

public function addReview()
{
    $this->uri = "/v2/book/reviews";
    $this->header = $this->authorizeHeader;
    // API默认请求设置为GET，因此这里需说明请求类型
    $this->type = 'POST';
    return $this;     
}
