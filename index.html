Simple-douban-oauth2

A simple php douban oauth2 client

Download this project as a .zip file Download this project as a tar.gz file

About

Simple douban oauth2为PHP开发者提供了一个非常简洁豆瓣api客户端。虽然暂时还不支持所有的api请求，但是你可以非常方便的添加自己需要的api。我也会尽快完成对所有api的支持，并逐步完善PHP客户端文档。

Status

目前支持的部分API有：


        - 用户 User
        - 读书 Book
        - 广播 Miniblog

Usage

这里提供了一个使用Simple douban oauth2来发送图片广播的例子。

Simple douban oauth2的所有API类都保存在api文件夹中，你可以参考一下方式非常方便的添加API：

添加无需授权的GET请求

public function get($id)
{
    $this->uri = '/v2/user/'.$id;

    return $this;
}

添加需授权的GET请求

public function me($accessToken)
{
    $this->uri = '/v2/user/~me';

    // 必须在header中发送授权令牌
    $this->header = array('Authorization: Bearer '.$accessToken);

    return $this;
}

添加需授权的POST请求

public function add($accessToken)
{
    $this->uri = "/v2/book/reviews";

    $this->header = array(
            'Content_type: application/x-www-form-urlencoded',
            'Authorization: Bearer '.$accessToken
            );

    // API默认请求设置为GET，因此这里需说明请求类型
    $this->type = 'POST';

    return $this;     
}
