Simple-douban-oauth2

A simple php douban oauth2 client

Download this project as a .zip file Download this project as a tar.gz file

About

Simple douban oauth2为PHP开发者提供了一个非常简洁豆瓣api客户端。虽然暂时还不支持所有的api请求，但是你可以非常方便的添加自己需要的api。我也会尽快完成对所有api的支持，并逐步完善PHP客户端文档。

Status

目前支持的部分API有：


        - 用户 User
        - 读书 Book
        - 广播 Miniblog
